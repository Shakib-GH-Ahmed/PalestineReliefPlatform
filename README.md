# Palestine Relief and Rehabilitation Initiative (PRRI)

A Django platform for managing relief and rehabilitation efforts in Palestine.

## Features
-Can be: Donator, Organizer, Volunteer
- Track aid distribution and donations
- Manage volunteers and beneficiaries

## Tech Stack
- Backend: Django
- Frontend: HTML, CSS, JavaScript
- Database: SQLite

## Installation
1. Clone the repo
2. Setup virtual environment
3. Install dependencies
4. Run migrations
5. Start server
